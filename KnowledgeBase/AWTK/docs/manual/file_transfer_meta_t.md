## file_transfer_meta_t

### 概述

## 文件元数据包(sender->receiver)。

### 属性

属性名称 | 类型 | 说明  
---|---|---  
name | char* | 即将传输文件的名称。  
size | uint32_t | 即将传输文件的长度。  
type | uint32_t | 包的类型。  
  
#### name 属性

* * *

> 即将传输文件的名称。

  * 类型：char*

特性 | 是否支持  
---|---  
可直接读取 | 是  
可直接修改 | 是
