## debugger_resp_t

### 概述

## 调试器响应头。

### 属性

属性名称 | 类型 | 说明  
---|---|---  
code | uint32_t | 响应码。  
error | uint32_t | 错误码(定义同ret_t)。  
size | uint32_t | 数据长度(除请求头外)。  
version | uint32_t | 版本号。  
  
#### code 属性

* * *

> 响应码。

  * 类型：uint32_t

特性 | 是否支持  
---|---  
可直接读取 | 是  
可直接修改 | 否
