## file_transfer_data_t

### 概述

## 文件数据包(sender->receiver)。

### 属性

属性名称 | 类型 | 说明  
---|---|---  
crc | uint32_t | 本次传输文件数据的CRC。  
offset | uint32_t | 本次传输文件数据的偏移量。  
size | uint32_t | 本次传输文件数据的长度。  
type | uint32_t | 包的类型。
