## debugger_req_t

### 概述

## 调试器请求头。

### 属性

属性名称 | 类型 | 说明  
---|---|---  
code | uint32_t | 请求码。  
data | uint32_t | 数据。  
size | uint32_t | 数据长度(除请求头外)。  
version | uint32_t | 版本号。  
  
#### code 属性

* * *

> 请求码。

  * 类型：uint32_t

特性 | 是否支持  
---|---  
可直接读取 | 是  
可直接修改 | 否
