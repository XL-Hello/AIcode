# 设备控制

## 设备控制 (deviceCtrl)

### 描述
用于控制设备，包括设置属性和执行功能。

### 子意图
- setProp: 设置属性
- execFunc: 执行功能

### 请求字段
- intent: "deviceCtrl"
- subIntent: "setProp" 或 "execFunc"
- targetId: 目标设备ID
- properties: 属性列表 (for setProp)
- function: 功能名称 (for execFunc)
- params: 参数

### 响应字段
- result: 执行结果

### 示例
设置属性：
```json
{
  "reqId": "dsadfaadsfvasfsa",
  "deviceId": "1748328523123",
  "uid": "9c5ca87258dd4057a28ee49888552895",
  "ctrlType":"setProp",
  "properties": {
    "onoff": "on"
  },
  "sourceType":"app",
  "sourceId": "1748328523125"
}
```

设置命令：
```json
{
  "reqId": "dsadfaadsfvasfsa",
  "deviceId": "1748328523123",
  "uid": "9c5ca87258dd4057a28ee49888552895",
  "ctrlType":"execFunc",
  "commands": {
    "name": "addDev",
    "args":{
    "number":250
    }
  },
  "sourceType":"app",
  "sourceId": "1748328523125"
}
```

相应参数：
Topic: v1/1748328523125/deviceCtrl

