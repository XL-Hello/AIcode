# 设备报告

## 属性报告 (reportProp)

### 描述
设备上报属性信息。

### 请求字段
- intent: "reportProp"
- deviceProps: 设备属性数组

### 示例
```json
{
  "reqId": "dsadfaadsfvasfsa1",
  "deviceProps":[
      {
          "deviceId":"1748328523123",
          "online":1,
          "uid": "9c5ca87258dd4057a28ee49888552895",
          "deviceType": "light",
          "timestamp": 1747376067607,
          "properties": {
            "onoff": "on"
          }
      },
      {
          "deviceId":"174832852322",
          "online":1,
          "uid": "9c5ca87258dd4057a28ee49888552895",
          "deviceType": "light",
          "timestamp": 1747376067607,
          "properties": {
            "onoff": "on"
          }
      }
  ],
  "sourceType":"device",
  "reportAll":true,
  "sourceId": "9c5ca87258dd4057a28ee49888552895"
}
```

## 事件报告 (reportEvent)

### 描述
设备上报事件信息。

### 请求字段
- intent: "reportEvent"
- events: 事件数组

### 示例
```json
{
  "reqId": "dsadfaadsfvasfsa1",
  "deviceId":"1748328523123",
  "uid": "9c5ca87258dd4057a28ee49888552895",
  "deviceType": "light",
  "timestamp": 1747376067607,
  "event": {
      "name": "upgradeStatusChanged",
      "args":{
        "upgradeStatus":"downloading",
        "value": "keyAction"
       }
  },
  "sourceType":"device",
  "sourceId": "9c5ca87258dd4057a28ee49888552895"
}
```
